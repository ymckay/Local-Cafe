# Local Cafe Website

## Overview
I built this project to practice responsive web design using HTML, CSS, and Bootstrap.

## Screenshot
<img src="./img/screenshot-home.png" width="600" alt="Local Cafe homepage">

## Features
- Responsive layout
- Clean and simple UI
- Multiple sections (home, menu, contact)
- Contact form UI (not connected to a backend)

## Tech Stack
- HTML
- CSS
- Bootstrap

## Live Demo
[View Live Demo](https://local-cafe.pages.dev/)

## What I learned
- Building responsive layouts with Bootstrap
- Structuring multi-page websites
- Creating a clean and consistent UI
- Improving accessibility with semantic HTML and aria labels

## Notes
This project focuses on layout and UI design.  
JavaScript functionality is minimal, and the contact form is not connected to a backend.
